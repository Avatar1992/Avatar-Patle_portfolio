Telecom NOC ETL Dashboard

Flask · PostgreSQL · Docker · Grafana · Terraform

Problem

Multiple OSS systems produced scattered alarms, lacking centralized visibility. Manual triage slowed incident response.

Solution

Built containerized ingestion services to normalize alarm data into PostgreSQL, powering Grafana dashboards and alerts. Infrastructure provisioned by Terraform and deployed via CI pipelines.

Impact

  • Centralized visibility and dashboards for NOC teams.
  • Faster triage and reduced incident response time (example: ~40% improvement).
